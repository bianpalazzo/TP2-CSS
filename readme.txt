TP2- Desarrollo de Software
Este repositorio contiene el código HTML y CSS para el Trabajo Práctico de Desarrollo de Software.

Descripción 
El proyecto consiste en una página web estructurada con HTML y estilizada con CSS. La página incluye varias secciones que demuestran diferentes aspectos del desarrollo front-end. El diseño incluye una variedad de estilos y efectos, tales como cambios de color, animaciones, y diseños responsivos. A continuación, se detallan las principales características y estilos aplicados:  propiedades de texto, propiedades de caja, posición Relativa Y Posición Absoluta ,media queries, principios de grid principios de animación. 


1.	Características

Estilos de Fuente y Color:
o	Uso de la fuente 'Courier New', Courier, monospace en todo el cuerpo del documento.
o	Colores específicos para el fondo del cuerpo y el encabezado.
2.	Estilos del Encabezado:
o	Fondo del encabezado con un color específico.
3.	Bloques de Contenido:
o	Bloques con fondo blanco, padding, margen y bordes redondeados.
o	Títulos dentro de los bloques con un borde inferior sólido.
4.	Cajas y Posicionamiento:
o	Cajas con bordes, sombras y colores específicos.
o	Uso de posicionamiento relativo y absoluto para ubicar elementos de manera precisa.
5.	Diseño Responsivo:
o	Aplicación de cambios de estilo para pantallas con un ancho máximo de 768px, incluyendo cambios en el tamaño de fuente y color de fondo.
6.	Grid Layout:
o	Uso de un contenedor grid para organizar elementos en columnas flexibles.
7.	Animaciones:
o	Animaciones simples de cambio de color al pasar el cursor sobre elementos.
o	Animaciones de movimiento que mueven elementos de izquierda a derecha y viceversa.
8.	Media Queries:
o	Implementación de media queries para adaptar el diseño a diferentes tamaños de pantalla.
9.	Transiciones y Transformaciones:
o	Uso de transiciones para suavizar cambios de estilo.
o	Transformaciones en animaciones para mover elementos en la página.
